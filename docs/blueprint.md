# **App Name**: MatchMingle

## Core Features:

- User Profiles: Create and manage user profiles with photos, bios, and interests.
- Profile Matching: Match potential partners based on mutual interests.
- Messaging: Send and receive messages with matched users.
- Location Based Matching: Show user profiles nearby
- AI Assisted Profile Tool: Use AI to suggest profile improvements for the user.
- Profile Storage: Use a Cloud SQL database to persist user profiles

## Style Guidelines:

- Primary color: Soft Lavender (#D0B8E3) for a gentle and inviting feel.
- Background color: Light Gray (#F2F0F4), offering a neutral canvas that makes profile content pop.
- Accent color: Dusty Rose (#C8A2C8) to highlight interactive elements, providing warmth and contrast against the cool primary and background.
- Headline font: 'Belleza' sans-serif for an artistic and stylish aesthetic, ideal for headers and titles.
- Body font: 'Alegreya' serif for readability and an elegant, intellectual feel, to make body content pleasant to read.
- Use simple, modern icons to represent different interests and preferences.
- Smooth transitions and subtle animations for a fluid and engaging user experience.
